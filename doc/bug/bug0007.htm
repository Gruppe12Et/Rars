Windows: Cars are grey in OpenGL.
 
Version 0.72
Solved in 0.73c
File g_view3d.cpp
Owner Marc Gueury
Platform Windows Only

Description:

The colors of the cars are grey when rars is run in OpenGL.

This is due to a missing line in the g_view3d.cpp.

In gview.cpp, in the function refresh().
