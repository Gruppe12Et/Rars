Movie

 
To replay a movie file:

On Unix

1) Copy the 2 movie files in your $RARS_HOME directory

  • movie.xy
  • movie.ang
2) Run Rars with a replay option (-mp<movie name>) and with the right track
    For the example in the directory movies: <adelaide.trk>.

   Ex:
       cp movies/movie.* .
       rars 8 80 adelaide -mpmovie

On Windows

1) Choose the name of the movie in the movie combo-box.
     The program shows all the *.xy files in the sub-directory "movies"
2) Choose the right track name in the track combo-box.
