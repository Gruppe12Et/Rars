Bug 0010

Windows: Rars crashes when replaying a movie with more than 8 cars
 
Version 0.73c
Solved in 0.73d
File g_vmanag.cpp
Owner Marc Gueury
Platform Windows Only
Impact on cars NONE
Impact on portability NONE

Description:

Rars crashes when replaying a movie with more than 8 cars if the value in the starting dialog box was left to 8.

WA: Use a more than 8 cars when starting.

Solution

This was a bug in g_vmanag.cpp. The procedure used car_count in place of MAXCARS during initialisation.
